INPUT in PYTHON

SINGLE VALUE INPUT :

var_name = input() // byDefault it takes STRING input

var_name = dataType(input())

MULTIPLE INPUT in PYTHON :

for i in range(n):

var_name = dataType(input())

MULTIPLE INPUT in SAME LINE in PYTHON :

var1,var2,var3,... = map(dataType, input().split())

MULTIPLE INPUT in SAME LINE into LIST in PYTHON :

list_name = list(map(dataType, input().split()))

KEY-VALUE PAIR INPUT in PYTHON :

var1, var2 = map(dataType, input().split())

myDict = {var1: var2}
